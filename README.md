## О проекте

Система для учета партнеров компании "Мастер пол", занимающейся производством напольных покрытий. Приложение позволяет вести базу партнеров, отслеживать историю продаж и автоматически рассчитывать скидки на основе объема реализации.

**Разработано в рамках производственной практики ПП.02.01**

### Основные возможности:
- Просмотр списка партнеров в виде карточек
- Добавление нового партнера
- Редактирование данных существующего партнера
- Удаление партнера
- Просмотр истории продаж выбранного партнера
- Автоматический расчет скидки (0% / 5% / 10% / 15%)
- Интуитивно понятный интерфейс с меню

## Быстрый старт

### 1. Требования к системе
- Windows 10/11
- Visual Studio 2022 или новее
- PostgreSQL 15+ (рекомендуется 17)
- .NET 10.0 SDK

### 2. Настройка базы данных PostgreSQL

#### Способ 1: Через pgAdmin (рекомендуется)
1. Откройте **pgAdmin**
2. Создайте новую базу данных с именем **Sadiullina**
3. Откройте **Query Tool** для этой базы
4. Скопируйте содержимое файла `database.sql` и выполните его
5. База данных создастся со всеми таблицами и тестовыми данными

#### Способ 2: Через командную строку
```bash
psql -U postgres -f database.sql
(Введите пароль от postgres при запросе)
